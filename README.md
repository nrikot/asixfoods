## ✨ Features

### Core Features
- 🏠 **Hero Section** with brand introduction and direct WhatsApp call-to-action
- 📦 **Product Catalog** displaying all products with images, prices, and descriptions
- 🔍 **Product Filtering** by categories (Cookies Premium, Gurih & Rempeyek, Olahan Kering)
- 🔎 **Search Products** real-time filtering by product name or description
- 🛒 **WhatsApp Integration** — every product has a direct "Order via WA" button
- ⭐ **Customer Testimonials** to build trust and social proof
- 📍 **Shipping & Payment Info** clear explanation of delivery options and payment methods
- ❓ **FAQ Section** accordion-style frequently asked questions
- 🎁 **Custom Order Form** for hampers, souvenirs, and bulk orders
- 📱 **Fully Responsive** works perfectly on mobile, tablet, and desktop
- 🌐 **Social Media Links** to Instagram, TikTok, and Facebook
